# 🛰️ Sentinel | Execution Intelligence

**Sentinel** is a high-fidelity project management command center designed for precision, visibility, and "Execution Intelligence." Built on a premium **Glassmorphism** design system, it transforms complex project data into an interactive, immersive "Glass Command" experience.

---

## ✨ Core Features

### 🧊 Glass Command Dashboard
An edge-to-edge, ultra-premium interface featuring a modular layout with high-contrast glass effects, dynamic blurs, and sophisticated micro-animations.

### 🌅 Morning Brief
Your daily situational awareness. A dedicated intelligence gathering view that surface critical updates, risks, and progress summaries before you dive into the details.

### 🌌 Immersive Dependency Graph
A full-page, interactive network visualization of your project's ecosystem.
- **Zen Pan Mode**: Double-click to "fly" through your data with cursor-following movement.
- **Scroll-Free Focus**: Strictly edge-to-edge interactive area with zero browser clutter.
- **Precision Zoom**: Cursor-centered magnification for detailed inspection.

### 💊 Operational Health Pill
A persistent, scroll-aware navigation and status companion. It intelligently hides when you need focus and surfaces when you need control, all while maintaining perfect backdrop-blur synchronization.

### 🛠️ Execution Management
Streamlined dedicated views for:
- **Alerts & Risk**: Real-time identification of project blockers.
- **Integrations**: Seamless connections to Slack, Gmail, Jira, and more.
- **Team Access**: Granular management of workspace members.

---

## 🛠️ Technology Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Iconography**: [Lucide Icons](https://lucide.dev/)
- **Design System**: Proprietary "Glass Command" Glassmorphism UI

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1.  **Clone and Install**:
    ```bash
    git clone [repository-url]
    cd sentinel
    npm install
    ```

2.  **Environment Setup**:
    Create a `.env.local` file and add your Gemini API Key:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```

3.  **Launch the Command Center**:
    ```bash
    npm run dev
    ```
